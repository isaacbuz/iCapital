Privacy Policy

Last Updated: January 2025

Important: This is a demonstration application created for the iCapital Solutions Engineer challenge. Please use test data only.

1. Information We Collect

This demonstration application does not directly collect or store any personal information. However, when you use the identity verification feature:

  • Information is collected directly by Parallel Markets through their SDK
  • Data is transmitted to Parallel Markets' sandbox environment
  • Console logs may temporarily display information for debugging purposes

2. How Information is Used

Any information entered is used solely for:

  • Demonstrating the identity verification workflow
  • Testing the integration with Parallel Markets SDK
  • Displaying results in the browser console for technical evaluation

3. Data Storage

This demonstration application:

  • Does not maintain any database
  • Does not store user information
  • Does not use cookies beyond what the Parallel Markets SDK may require
  • Only displays information temporarily in the browser console

4. Third-Party Services

This integration uses the Parallel Markets SDK, which has its own privacy practices. When you interact with the identity verification overlay, you are subject to Parallel Markets' privacy policy.

5. Security

As a demonstration application:

  • Uses HTTPS when deployed
  • Connects only to Parallel Markets' sandbox (demo) environment
  • Does not process or store sensitive data

6. Your Rights

Since this application does not store personal data, there is no data to request, modify, or delete. Any data submitted through the Parallel Markets overlay is governed by their policies.

7. Test Data Only

Users should only enter test or fictional data in this demonstration. Do not enter real personal information, social security numbers, or other sensitive data.
